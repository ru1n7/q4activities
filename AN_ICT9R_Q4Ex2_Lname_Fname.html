Introduction to the Three Types of Javascript Outputs

Three Types of Javascript Output

  • innerHTML

    • The innerHTML property sets or returns the HTML content (inner HTML) of an element. It can be combined with document.getElementByID

  • document.write
  • Window Alert

    • An alert box is often used if you want to make sure information comes through to the user. It pops up and the user has the click "OK" to be able to proceed to the site
